# Guia Turístico – Brasil Incrível

Este projeto apresenta um website informativo sobre destinos turísticos do Brasil. O site é construído utilizando apenas **HTML**, oferecendo uma estrutura organizada e semântica para quem deseja conhecer mais sobre as belezas naturais, culturais e geográficas do país.

## Funcionalidades

* **Apresentação do Brasil:** O site introduz o usuário às principais características do país, destacando sua diversidade cultural e natural.
* **Organização por regiões:** As regiões turísticas são apresentadas de forma estruturada, facilitando a compreensão da divisão geográfica do Brasil.
* **Lista de motivos para visitar o país:** Exibe tópicos com razões que tornam o Brasil um destino turístico atrativo.
* **Tabela comparativa:** Inclui uma tabela com características das regiões, como paisagens, cultura e clima, organizadas de forma clara.
* **Galeria de imagens:** O site apresenta imagens representando destinos brasileiros, tornando a navegação mais visual e atrativa.
* **Vídeo incorporado:** Possui um vídeo do YouTube incorporado diretamente na página, enriquecendo a experiência do usuário.
* **Estrutura semântica:** Utiliza corretamente elementos como `header`, `main`, `section`, `article`, `aside` e `footer`, garantindo boas práticas em HTML5.

## Tecnologias Utilizadas

* **HTML:** Utilizado para estruturar todo o conteúdo do site, aplicando boas práticas de semântica e organização.
